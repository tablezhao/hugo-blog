组装 Intel NUC 指南

October 26, 2018 ·  · · 极客

由头

买一台新电脑的想法来源于买了显示器之后，每天需要将笔记本搬来搬去很麻烦，我是一个很怕麻烦的人，所以意识到需要做一点改变来解决这些麻烦。最开始我其实是想买Mac Mini，搜了下亚马逊发现翻新版的Mac Mini最便宜的是不到400美元，可是这毕竟是4-5年前发布的产品了，看到网上有些评测说连最新版的 OS X都快跑不动了，所以买一个全面落后的产品是不可能的。

放弃Mac Mini之后，我把目光转向了 Lenovo Thinkcentre M710 Tiny 和 M715 Tiny这两款机器，然而M710 Tiny价格基本上在600刀以上，M715 Tiny不仅是AMD的CPU，而且还缺货了好久，所以也只能将这两款放弃了。

放弃知名品牌的机器之后，我想是不是看一看一些小众品牌的机器，比如很多我都没听说过名字的 Mini PC，但在Youtube 看了很多评测之后，发现类似的机器质量似乎都有些不可靠，而且扩展性不高，今后想要升级就很困难了。

至此，我终于发现了Intel 的亲儿子 NUC Kit 系列，拥有5代至8代 i3至 i7全系列CPU，可以自己扩展内存和硬盘空间，既可以组装一台性能均衡、长期使用的Mini PC，又可以搞一台接近发烧级别的全能PC，除此之外，还有三年的全球联保，价格也挺亲民，简直是完美了。

Shut UP， Take My Money！

备材

在确定购入 Intel NUC之后，需要做的事情就简单了。首先去买 Intel NUC Kits i3BNH 这个套件，买7代i3的NU的理由是——打折！换算下来比国内购入要便宜了25%左右。除此之外，根据我的使用需求，CPU不是一个关键且核心的点，于是7代i3版本对我来说是个很好的选择。（目前计划回国以后这个套件给我爸妈用，然后重新组装一台8代/9代 i5版本的NUC自用。）

因为 Kits 版本是不含 Memory 及 Storage 的，所以在买 NUC Kits 的同时，我买了一根 8G DDR4 2400 SOIDMM 的 Memory，一块 256G M.2 SSD，这样我就买齐了组装一台 Intel NUC的配件。

PS：显示屏是 HP VH240a，之前是为了投射笔记本用，现在正好拿来接 NUC。

组装

因为NUC Kits 其实就是一块主板及一些零部件，所以买回来的Memory及Storage需要自己安装到这块主板上去。好在我很早以前就有过拆装电脑零件的经验了，正好手头也有螺丝刀，于是就自己动手开始组装。

首先是打开 NUC Kits 的后盖，可以看到一个 2.5 寸的硬盘托，把硬盘托取下才能看到主板的全部面貌。从这个角度看过去，可以看到有两个 Memory 的接口，以及一个 M.2 SSD 的接口，于是我把买来的 Memory 拆开插到卡槽里，再把 SSD 拆开并插到相应的卡槽里，最后把硬盘托复位，合上后盖并拧紧螺丝，硬件组装部分就完成了！

开机&装系统

硬件组装好之后，就可以接通电源开机了。好久没用台式机，所以看到BIOS界面还是觉得很新鲜的。进入BIOS检查了下各项设置之后，就插上已经制作好成为Windows 10系统安装介质的U盘，然后重新启动NUC，就会进入系统安装界面了。

Windows 10安装起来很简单，按部就班即可，不过记得要支持正版哦~

Done！

系统安装和激活完，一台全新可用的NUC就在我面前了。再安装上必备的Dropbox和Google Drive这些工具，这台NUC就可以用于学习/工作了，一切大功告成！
